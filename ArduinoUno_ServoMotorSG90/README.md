# Arduino Uno Servo Motor Control (SG90)

This project demonstrates basic control of an SG90 servo motor using Arduino Uno. The servo rotates to different angles based on programmed instructions and can be extended for automation and robotics applications.

## Features

- Servo motor control using Arduino Uno  
- Precise angle positioning (0° to 180°)  
- Simple and beginner-friendly setup  
- Smooth motion using PWM signal  
- Suitable for robotics and automation projects  

## Components Required

- Arduino Uno  
- SG90 Servo Motor  
- Breadboard (optional)  
- Jumper wires  
- USB cable  

## Circuit Connections

### SG90 Servo Motor
- Red   -> 5V  
- Brown -> GND  
- Orange -> Digital Pin 9 (PWM)  

## Working

- The Arduino Uno sends PWM signals to the servo motor.  
- The servo rotates to specified angles based on the signal.  
- Typical positions include:
  - 0 degrees  
  - 90 degrees  
  - 180 degrees  
- The servo continuously moves between positions in a loop.  
- This working principle is widely used in robotics, automation, and control systems.
